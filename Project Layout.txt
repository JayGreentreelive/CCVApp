Project Layout
	CCVApp.Shared
		Code that is specific to CCVApp but used across both iOS and Android.
	Droid
		Android specific CCVApp code
	iOS
		iOS specific CCVApp code
	
	Rock.Client.Models
		Models from the Rock project
	
	Rock.Mobile
		Collection of platform agnostic code that can be reused in any app.
			(Similar to CCVApp.Shared, except not specific to CCVApp)
